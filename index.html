Web Control

Control my mac through the web!
Enter any shell command here and if the daemon is running, the command wil be executed.

How it works

The commands given here go to a firebase database which acts as a queue. If the daemon is running, the commands are read in a sequential manner. If it has been executed, the listing is cleared and the webpage is updated. You might have noticed an obvious security flaw in it. I hope that you don't give malicious commands :)
